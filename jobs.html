Jobs at Qiscus

Front-End Web Developer.

Requirements :

  • Menguasai HTML, CSS, Javascript.
  • Mempunyai pengetahuan tentang back-end server.
  • Berpengalaman dengan Javascript framework (jQuery, Backbone.js, Knockout.js).
  • Mempunyai pengetahuan tentang AJAX.

Deskripsi dan Benefits :

  • Bekerja dengan tim untuk develop front-end web aplikasi Qiscus.
  • Men-develop web interface (UI dan UX).
  • Mengintegrate front-end tersebut dengan sistem back-end yang dikerjakan oleh tim back-end server.
  • Basic salary mulai dari 3,5 juta rupiah/bulan (full-time position), mulai dari 1,5 juta rupiah/bulan (part-time position).

Mobile Developer (Android).

Requirements :

  • Menguasai Android Programming.
  • Memiliki pengetahuan tentang aplikasi Android yang terkoneksi dengan server web.
  • Berpengalaman dengan Aplikasi android yang sudah sampai tahap production stage.

Deskripsi dan Benefits :

  • Bekerja dengan tim untuk develop aplikasi Android Qiscus.
  • Basic salary mulai dari 3,5 juta rupiah/bulan (full-time position), mulai dari 1,5 juta rupiah/bulan (part-time position).
